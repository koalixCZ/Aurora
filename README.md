# Aurora
## Co to je?
Aurora (řecky Éós) byla bohyní úsvitu ranních červánků. Byla krásná a milovala všechno krásné, především krásné muže.
## Jak mi to může pomoci?
Člověk by se měl učit nové věci.
## Jak to funguje?
Stačí chtít.
